== Description ==

This plugin places a simple JavaScript free back to top button on the footer of your site. It's a CSS only solution, no bloatware here, so fully compatible with <a href="https://amp-wp.org">AMP</a>. 

If you're using the official AMP plugin there are options to  display the button on AMP URLs only. 

<strong>Features: </strong>

<ul>
	<li>Easy to setup, activate and it's live!</li>
	<li>Super Lightweight</li>
	<li>Unlimited color Selection</li>
	<li>AMP Compatible</li>
	<li>JavaScript free</li>
	<li>No annoying popups, notices or suggestions</li>
</ul>

== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as a typical WordPress plugin (from the WordPress repository or via a file upload)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. After activate plugin you will see "CSS Only Scroll to Top" menu at left side on WordPress dashboard.
4. Once activated you can access the settings tab under "Settings > CSS Only Scroll to Top"

== Frequently Asked Questions ==

= Does this plugin come with settings? =

There are colour settings along with AMP display settings. For future releases there will be opactity options, along with positioning settings. 

= What advantages does this plugin have over other back to top buttons? =

It's JavaScript free, with minimal CSS styling. It therefore has little impact on page site performance. It's also fully AMP compatible. 

= I'm using AMP in Reader mode, will this back to top button work? =

Yes, it works in all modes of the <a href="Update to readme">official AMP plugin</a>.
